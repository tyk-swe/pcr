===== packetcraftr --help =====
PacketcraftR builds and dissects arbitrary packet stacks with exact bytes, bounded parsing, passive route planning, and policy-gated live workflows. Native features, dependencies, and privileges determine which live paths are available.

Usage: packetcraftr [OPTIONS] <COMMAND>

Commands:
  build       Build exact packet bytes from an expression or document
  dissect     Decode a frame with bounded, registry-driven dissection
  read        Stream frames from a classic PCAP or PCAPNG file
  interfaces  Enumerate local interfaces
  plan        Passively select route, source, MTU, and link mode
  send        Transmit a packet under traffic policy
  exchange    Capture-ready request/response exchange
  capture     Stream live captured frames
  replay      Replay a PCAP/PCAPNG stream
  scan        Run a structured network scan
  traceroute  Run structured traceroute probes
  dns         Run a structured DNS operation
  fuzz        Run bounded field-aware packet fuzzing
  routes      Enumerate passive interface-bound route decisions
  help        Print this message or the help of the given subcommand(s)

Options:
      --output <OUTPUT>
          [default: text]
          [possible values: text, json, ndjson, hex, raw, pcap, pcapng]

  -h, --help
          Print help (see a summary with '-h')

  -V, --version
          Print version
