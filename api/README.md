# Versioned public API evidence

`packetcraftr-v0.2-beta.txt` is the machine-readable v0.2 beta façade baseline.
It records public paths, declarations, and public inherent/trait members from
rustdoc generated by the pinned Rust 1.96 toolchain. Its header contains the
SHA-256 digest also recorded in `CHANGELOG.md`.

The human contract is `docs/public-api.md`. Run:

```console
RUSTDOCFLAGS='-D warnings' cargo doc --locked --workspace --no-deps
python3 scripts/check-public-api.py
```

to prove that the current generated reference matches this versioned baseline.
Both files and the verification script are included in the buildable GitHub
Release workspace archive. Updating the baseline requires an explicit
compatibility review and matching changelog digest; it is not a generated file
that ordinary tests may rewrite.
