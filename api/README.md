# Public API baseline

`packetcraftr-public-api.txt` records the public façade paths, declarations,
and inherent or trait members generated by the pinned Rust toolchain.

The human-facing contract is [`docs/public-api.md`](../docs/public-api.md).
Verify both with:

```console
RUSTDOCFLAGS='-D warnings' cargo doc --locked --workspace --no-deps
python3 scripts/check-public-api.py
```

Update the baseline only after reviewing the compatibility impact; ordinary
tests and documentation builds must not rewrite it.
